# 🚀 Alan的自驱进化计划

一个全面的学习管理系统，帮助小学生培养良好的学习习惯，实现自我驱动成长。

## ✨ 功能特点

### 📊 核心功能
- **总览面板** - 一目了然的学习数据统计
- **作业管理** - 记录和跟踪作业完成情况
- **阅读管理** - 书单管理和阅读进度跟踪（支持字数统计）
- **学习计划** - 制定和管理学习目标
- **学习测试** - 记录测试成绩，检验学习效果
- **运动提升** - 羽毛球和网球训练记录

### 🎯 特色功能
- **番茄闹钟** - 25分钟专注学习法
- **作息表** - 每日作息管理，自动循环重置
- **注意力训练** - 舒尔特方格、记忆翻牌、找不同游戏
- **积分系统** - 完成任务获得积分
- **积分兑换** - 兑换零花钱、游戏时间
- **探索世界** - 用5000积分解锁中国旅游城市

### 💰 积分获取规则
- 完成作业：+10积分
- 完成番茄钟：+5积分
- 读完一本书：+20积分
- 更新阅读进度：+2积分
- 完成学习计划：+8积分
- 完成作息项：+1积分
- 学习测试：+10~30积分（根据分数）
- 运动训练：+10~20积分（根据时长）

## 💾 数据存储

系统使用浏览器本地存储（localStorage），数据保存在您的设备上：
- ✅ 无需注册登录
- ✅ 数据完全私密
- ⚠️ 清除浏览器数据会丢失记录
- ⚠️ 不同设备无法同步

4. 设定合理的学习目标和奖励
